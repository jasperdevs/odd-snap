# Code of Conduct

Yoink is an open-source project. Treat other contributors and users with respect.

## Expected behavior

- Be direct without being abusive
- Assume good faith by default
- Keep feedback focused on the work, not the person
- Be careful with screenshots, logs, and examples that may expose private information

## Unacceptable behavior

- Harassment, hate speech, or threats
- Doxxing or sharing private information without consent
- Repeatedly derailing issues or pull requests
- Deliberately trying to make collaboration unpleasant

## Enforcement

Maintainers may close issues, hide comments, or reject contributions that violate this policy.

If you need to report a conduct issue, open a GitHub issue and mark it clearly so it can be handled privately.
